## C++ OOP Playground

This folder contains:

- Example `.cpp` files for each major C++ OOP concept.
- A single notes file (`oops_notes.md`) summarizing the theory with pointers to the example files.

You can compile any example with:

```bash
g++ -std=c++20 -Wall -Wextra -pedantic -O2 <file>.cpp -o <file>
./<file>
```
