| Term          | Definition                                                            |
| ------------- | --------------------------------------------------------------------- |
| **Substring** | Any contiguous part of a string. `"abc"` in `"abcd"` is a substring.  |
| **Prefix**    | A substring starting from index 0. `"ab"` is a prefix of `"abcd"`.    |
| **Suffix**    | A substring ending at the last index. `"cd"` is a suffix of `"abcd"`. |
